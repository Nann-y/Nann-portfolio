## Local Media Folders

Put your portfolio media files in these folders:

- `public/assets/projects/images`
- `public/assets/projects/videos`
- `public/assets/others/images`
- `public/assets/others/videos`

You can reference them in code with paths like:

- `/assets/projects/images/my-project-cover.jpg`
- `/assets/projects/videos/my-project-demo.mp4`
- `/assets/others/images/my-other-work.png`
- `/assets/others/videos/my-other-work.mp4`
